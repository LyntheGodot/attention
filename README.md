# 注意力番茄钟 · Attention is all you need!

> 一款专为专注力低下、ADHD人群打造的随机番茄钟应用，利用心理学效应让专注变得轻松有趣。

## ✨ 产品定位

市面上多数番茄钟应用过于死板，缺乏针对专注力障碍人群的人性化设计。我们的产品以**随机反馈心理学效应**为核心，打破传统固定时长的束缚，通过多样化的趣味化反馈、优质的交互设计和创新功能，帮助用户轻松进入心流状态，培养长期专注习惯。

### 核心优势
- 🎲 **随机机制**：利用不确定的正向反馈促进多巴胺分泌，降低专注的心理负担
- 🧘 **正念导向**：微休息时间引导注意力回收，培养自我感知能力
- 🎨 **体验优先**：简洁优雅的界面设计，无广告、无冗余功能
- 📊 **数据驱动**：个性化的专注数据统计和分析，让进步看得见
- 🔧 **高度自定义**：灵活适配不同人群的专注习惯和需求

## ✅ 已实现功能

### 🎯 核心功能
- 🍅 **可视化进度**：优雅的进度圆环实时显示专注进度
- 🔔 **随机提示**：自定义时间范围内随机播放提示音，提醒收回注意力
- 🌿 **微休息**：浅绿色界面引导短暂正念休息，降低认知负担
- 📈 **专注统计**：每日专注时间记录，带动画效果的柱状图可视化
- 🌲 **白噪音陪伴**：支持森林/雷雨两种白噪音，独立音量调节
- ⚙️ **灵活配置**：自定义专注/休息/间隔/微休息时长、提示音音量、白噪音类型

### 🎵 音频系统
- 双声道独立控制：提示音和白噪音互不干扰
- 支持自定义提示音次数
- 白噪音循环播放，支持实时暂停/继续
- 音量滑块实时调节，直观易用

### 💾 数据管理
- 本地JSON存储专注数据，隐私安全
- 自动记录每日专注时长
- 统计页面展示最近14天专注趋势

## 🎮 使用指南

### 快速开始
1. 点击「随机番茄钟」进入专注界面
2. 点击右上角⚙️图标可自定义各种参数
3. 点击「开始」按钮开始专注
4. 听到提示音后进行短暂微休息，收回你的注意力
5. 专注完成后可在「统计」页面查看专注记录

### 白噪音使用
- 计时开始后，点击「🌲 播放白噪音」即可开启背景音
- 白噪音会循环播放，可随时暂停/继续
- 在「设置-声音」中可切换白噪音类型（森林/雷雨）和调节音量

### 最佳实践
- 初期建议从25-30分钟专注时长开始，逐步增加
- 微休息时建议闭上眼睛，感受呼吸，不要看屏幕
- 随机间隔建议设置为专注时长的1/8 ~ 1/4，效果最佳
- 白噪音可以帮助屏蔽外界干扰，根据环境选择合适的类型

## 🧠 设计理念

本产品的核心设计基于以下心理学原理：
1. **可变比率强化**：随机时间的提示音和奖励机制，比固定间隔更能强化行为
2. **正念冥想**：微休息时间引导注意力回收，提升元认知能力
3. **最小摩擦力原则**：简洁的界面和操作，降低启动专注的心理门槛
4. **正向反馈循环**：数据统计和成就系统，让进步可视化，增强自我效能感
