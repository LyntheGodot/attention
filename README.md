# 注意力番茄钟

一个帮助你培养专注力的番茄钟应用。

## 功能特点

- 🍅 **可视化进度**：优雅的进度圆环显示专注进度
- 🔔 **随机提示**：在专注过程中随机时间点播放提示音，提醒你"收回注意力"
- 🌿 **微休息**：提示后显示浅绿色界面，引导你进行短暂的正念休息
- ⚙️ **灵活设置**：可自定义专注时间、休息时间、提示间隔和微休息时长

## 安装依赖

```bash
pip install pillow pygame
```

或者使用 requirements.txt：

```bash
pip install -r requirements.txt
```

## 运行应用

```bash
python main.py
```

## 使用说明

### 1. 主界面

- **随机番茄钟**：点击开始使用番茄钟功能
- **说明**：查看应用详细说明

### 2. 番茄钟界面

- **进度圆环**：实时显示专注进度
- **数字计时**：清晰的倒计时显示
- **控制按钮**：开始/暂停/重置计时
- **设置按钮**：右上角⚙图标可打开设置界面

### 3. 设置界面

#### 基础设置：
- **专注时间**：单次专注的持续时间（默认40分钟）
- **休息时间**：专注结束后的休息时间（默认5分钟）

#### 随机提示设置：
- **最小间隔**：提示音可能出现的最早时间（默认5分钟）
- **最大间隔**：提示音可能出现的最晚时间（默认40分钟）
- **微休息时间**：提示后微休息的持续时间（默认60秒）

**重要**：最大间隔不能超过专注时间！

### 4. 说明

当听到提示音时，会显示浅绿色背景的微休息界面，文字显示：
"现在，收回你的注意力，闭上眼睛去感知它"

持续微休息时间后，会再次播放提示音并返回番茄钟界面。

## 技术说明

- **Python 3.x**：编程语言
- **Tkinter**：内置GUI库
- **PIL/Pillow**：图像处理（进度圆环绘制）
- **Pygame**：音频播放（提示音）

## 应用理念

"Attention is all you need!" — 专注于当下，感知自我。这个应用结合了番茄工作法和正念休息的理念，帮助你在学习和工作中保持良好的专注力。
